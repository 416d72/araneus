# Dependencies

- Linux
- Python3.6
- `pkexec` or `kdesu` or `kdsudo` packages installed.

# Installation

> Instructions and examples will be available in the first stable release.

# Building

- `source env/bin/activate`
- `pip install -r requirements.txt`
- `cd Araneus/`
